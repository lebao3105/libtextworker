Customized Tkinter Text widget with a basic right-click menu.

Parameters
parent: Where to place this widget
useMenu(bool) : Enable right-click menu or not
unredo(bool)=True : Undo Redo
useScrollbars(bool)=True : Use scrollbars
**kwds: Other configurations (tkinter.Text)

You can set TextWidget.wrapbtn to your own wrapbtn to use the wrap feature. The wrap function is wrapmode(event=None).
