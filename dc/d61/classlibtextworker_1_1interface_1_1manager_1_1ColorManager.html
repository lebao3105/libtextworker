libtextworker.interface.manager.ColorManager.GetFont (   self)

Property of ColorManager to call the font definitions.

When called, this returns the following: (font) size (int|str), style, weight, family It will vary on different GUI toolkits:

  • wxPython: wx.Font object
  • Tkinter: tkinter.font.Font object
